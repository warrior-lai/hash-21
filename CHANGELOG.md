# Changelog

All notable changes to Hash21 will be documented in this file.

## [0.3.0] - 2026-03-25

### Added
- User management system with role-based access (admin/artist/collaborator)
- Auto-create artist when adding user with artist role
- sitemap.xml and robots.txt for SEO
- CI/CD with GitHub Actions

### Changed
- Admin panel: artists-only view for non-admins
- Improved certificate section styling in admin

### Fixed
- Autocomplete disabled on user forms
- JSON syntax in hackathon submission

## [0.2.0] - 2026-03-24

### Added
- On-chain certification with OpenTimestamps
- Certificate PDF generation
- Verify page for public verification
- Admin panel with full CRUD
- XSS sanitization across all forms

### Changed
- Dashboard shows certificates, zaps, sats stats
- README with badges and architecture

## [0.1.0] - 2026-03-07

### Added
- Initial release
- Lightning zaps (NIP-57)
- WebLN support (Alby)
- Shop with LNURL-pay
- Dynamic gallery from Supabase
- Artists hub with Lightning Address
- Bilingual (ES/EN)
